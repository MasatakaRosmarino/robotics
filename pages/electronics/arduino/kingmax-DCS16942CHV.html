Experiment 1:

Equipment needed

  • 1.5V AA battery: X4
  • 4 AA battery holder
  • Jumper cables X4
  • kingmax-DCS16942CHV servo motor
  • Arduino uno
  • Arduino USB cable
Setup

drawing and schematics here...

sketch
                        #include <Servo.h>
                        
                        Servo myservo;  
                        int pos = 0;   
                        
                        void setup() {
                          myservo.attach(9); 
                        }
                         
                        void loop() {
                          for (pos = 0; pos <= 180; pos += 1) { 
                            myservo.write(pos);              
                            delay(15);                       
                          }
                          for (pos = 180; pos >= 0; pos -= 1) { 
                            myservo.write(pos);              
                            delay(15);                       
                          }
                        }
observation

  • Servo horn rotates from 0 to 180ยบ
