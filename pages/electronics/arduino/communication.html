communication

serial communication (uart)

ports are on pins:

  • 0, RX
  • 1, TX

additional UART ports can be added via software.

USB communication is not possible at the same time UART communication is in progress.

i2c (twi) communication

Ports are on pins:

  • scl and A5
  • sda and A4

up to 128 different devices in series can be connected to this ports (accelerometer + T sensor etc.).
each device is the slave, and must have a unique address to communicate with Arduino, the master.

SCL port acts as the clock.

SDA port acts as the data bus.

spi communication

it is a 4-wire communication

  1. MOSI (Master Out Slave In)
  2. MISO (Master In Slave Out)
  3. SCK, clock
  4. SS, Slave selector

connection to multiple devices is possible, but when one SS is set to HIGH, all others have to be set to LOW.
