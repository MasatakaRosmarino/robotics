sg-90 servo experiment

tools

servo motor hooked up to arduino uno pins:

  • 5v
  • gnd
  • digital pin 9 (˜)
sketch
                        #include <Servo.h>
                        
                        Servo myservo;  // create servo object to control a servo
                        // twelve servo objects can be created on most boards
                        
                        int pos = 0;    // variable to store the servo position
                        
                        void setup() {
                          myservo.attach(9);  // attaches the servo on pin 9 to the servo object
                        }
                        
                        void loop() {
                          for (pos = 0; pos <= 180; pos += 1) { // goes from 0 degrees to 180 degrees
                            // in steps of 1 degree
                            myservo.write(pos);              // tell servo to go to position in variable 'pos'
                            delay(15);                       // waits 15 ms for the servo to reach the position   
                          }
                          for (pos = 180; pos >= 0; pos -= 1) { // goes from 180 degrees to 0 degrees
                            myservo.write(pos);              // tell servo to go to position in variable 'pos'
                            delay(15);                       // waits 15 ms for the servo to reach the position
                          }    
                         }

servo 1 observations:

  • measured voltage from power source is 4.84V - 4.79
  • voltage from PWM signal goes from around 144 mV to around 576 mV
  • measured current from power source goes from around 0.35 to around 0.283 A
  • measured current from PWM signal goes from 0.01 mA to 0.03 mA
  • pronounced flickering when reaching pos 0 and 180
  • light flickering when reaching pos 180
  • when rotating from 180 to 0 it starts faster than the other way around
  • Max torque is less than 0.5 Kg (small water bottle) VS 1.8 Kgf/cm (declared stall torque)

servo 2 observations:

  • measured voltage is 4.84V
  • measured current goes from around 0.65 to around 0.257 A
  • light flickering when reaching pos 0 and 180
  • when rotating from 0 to 180 it starts faster than the other way around
  • the servo emits a buzzing sound
