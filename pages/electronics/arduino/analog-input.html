analog input experiment

setup

  • pc
  • arduino uno board
  • arduino-PC USB cable
  • breadboard
  • 10 KΩ 0.25W resistor X2
  • jumper wires X 3

source code

                    int analogIn = A0;
                    
                    void setup() {
                      Serial.begin(9600);
                    }
                    
                    void loop() {
                      Serial.println(analogRead(analogIn));
                      delay(500);
                    }

drawing of arduino and wired components here

Observation when using 5V output from Arduino

  • with 1 resistor, the reading is 0
  • with 2 10 Kohm resistors there is a reading of around 509

Observation when using 3.3V output from Arduino

  • with 1 resistor, the reading is 0
  • with 2 10 Kohm resistors there is a reading of around 340

the analogRead() function reads the input voltage in relation with the source voltage.

  • the lower the voltage, the closer it gets to 1023
  • the higher the voltage, the closer it gets to 0
