feetech-FT5330M servo experiment

tools

servo motor specs

  • Operating Voltage Range: 6V-7.4V
  • Idle current (at stopped): 6MA @ 7.4V
  • No load speed: 0.238 sec/60° @ 6V
  • No load speed: 0.192 sec/60° @ 7.4V
  • Runnig current (at no load): 180mA @ 7.4V
  • Peak stall torque: 28.8kg.cm @ 6V
  • Peak stall torque: 35.5kg.cm @ 7.4V
  • Rated torque: 11.8 kg.cm @ 7.4V
  • Stall current: 3.9A @ 7.4V
  • Stop position: 1500 sec
  • Running degree: 220° Continuous Rotation
  • Pulse width range: 500~2500 usec

Experiment 1:

Equipment needed

  • 1.5V AA battery: X4
  • 4 AA battery holder
  • Jumper cables X4
  • feetech-FT5330M servo motor
  • Arduino uno
  • Arduino USB cable
Setup

drawing and schematics here...

sketch
                                #include <Servo.h>
                                
                                Servo myservo;  
                                int pos = 0;   
                                
                                void setup() {
                                  myservo.attach(9); 
                                }
                                 
                                void loop() {
                                  for (pos = 0; pos <= 180; pos += 1) { 
                                    myservo.write(pos);              
                                    delay(15);                       
                                  }
                                  for (pos = 180; pos >= 0; pos -= 1) { 
                                    myservo.write(pos);              
                                    delay(15);                       
                                  }
                                }
observation
  • Battery pack (Energizer brand) voltage is 6.36V
  • Servo horn rotates from 0 to 180º

                                  myservo.write(0); 
                                  delay(1000);
                                
                                  myservo.write(60);
                                  delay(1000);
                                
                                  myservo.write(120); 
                                  delay(1000);
                                
                                  myservo.write(180); 
                                  delay(1000);
                                
                                  myservo.write(120); 
                                  delay(1000);
                                
                                  myservo.write(60);
                                  delay(1000);
                                }
observation
  • Battery pack (Energizer brand) voltage is 6.36V
  • Servo horn positions itself correctly at the angles indicated in the sketch by rotating at maximum speed allowed
