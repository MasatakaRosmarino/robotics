digital pins output voltage and current

setup

  • pc
  • arduino uno board
  • arduino-PC USB cable
  • 3mm blue LED
  • 3mm red LED
  • 3mm green LED
  • 3mm yellow LED
  • jumper wires X 2

source code

                    void setup() {
                      pinMode(0, OUTPUT);
                    }
                    
                    void loop() {
                      digitalWrite(0, HIGH);
                      delay(10000);
                      digitalWrite(0, LOW);
                      delay(10000);
                    }

voltage measurement with blue LED

  • when pin state is HIGH, voltage is around 3.39 V
  • when pin state is HIGH, voltage is around 0.104 V

  • when pin state is HIGH, current is around 0.0354 A
