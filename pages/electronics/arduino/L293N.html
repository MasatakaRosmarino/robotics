Determine output voltage

sketch
                                
                                    int motor1     = 8; 
                                    int direction1 = 9; 
                                        
                                    void setup(){ 
                                        pinMode(motor1, OUTPUT);  
                                        pinMode(direction1, OUTPUT);  
                                    } 
                                    
                                    void loop(){ 
                                        digitalWrite(motor1, HIGH);   
                                        digitalWrite(direction1, LOW);             
                                    }

Equipment needed

  • 1.5 V AA batteries x 6
  • 6 AA Battery holder
  • DC motor 130 size
  • Multimeter
  • Dupont jumper wires male-female

Procedure

  • Measure the voltage across the input voltage pins
  • Measure the output voltage pins

Observations

  • Measured batteries voltage: 9.37 V
  • Measured output voltage from L293N (motor 1): 4.03 V
