L239D Motor Shield

L293 board

Shield specs

  • Input voltage: 4.5 ~ 36 V
  • Maximum input-output current
  • Maximum DC motor connections: 4
  • Maxium servo motor connections: 2

Required libraries

Adafruit motor shield library: here

Experiment 1:

Equipment needed

  • 1.5V AA battery: X4
  • 4 AA battery holder
  • DC motor 3 ~ 6 V: x1
  • L239D motor shield
  • Arduino uno
  • Arduino USB cable
Setup

drawing and schematics here...

sketch
                        #include <Servo.h>
                        
                        Servo myservo;  
                        int pos = 0;   
                        
                        void setup() {
                          myservo.attach(9); 
                        }
                         
                        void loop() {
                          for (pos = 0; pos <= 180; pos += 1) { 
                            myservo.write(pos);              
                            delay(15);                       
                          }
                          for (pos = 180; pos >= 0; pos -= 1) { 
                            myservo.write(pos);              
                            delay(15);                       
                          }
                        }
observation
  • Battery pack (Energizer brand) voltage is 6.36V
  • Servo horn rotates from 0 to 180ยบ
