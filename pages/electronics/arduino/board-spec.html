boards types and specs

arduino UNO

  • ATmega328P ATMEL-based 8-bit-microcontroller
  • operating voltage is 5V
  • recommended input voltage is 7-12V. Max input voltage limit is 6-20V
  • total digital pins (r/w) are 20, 14 (0 to 13) + 6 also used for analog (A1 to A6)
  • analog pins store the voltage of the signal in 10 bits
  • 6 PWM pins
  • DC current per I/O pin is 20mA
  • DC current from the 3.3V pin is 50 mA
  • clock speed is 16 Mhz
  • memory size is 32 KB of which 0.5 KB are used by the bootloader
  • SRAM is 2 KB
  • powered by 2.1 mm center positive plug

arduino Mega 2560

  • ATMEL-based 8-bit-microcontroller
  • operating voltage is 5V
  • recommended input voltage is 7-12V. Max input voltage limit is 6-16V
  • total digital pins (r/w) are 54 of which 12 are also analog
  • 2 analog output pins (DAC)
  • 12 PWM pins
  • DC current per I/O pin is 130mA
  • DC current from the 3.3V pin is 800 mA
  • DC current from the 5V pin is 800 mA
  • clock speed is 16 Mhz
  • SRAM is 8 KB
  • memory size is 256 KB of which 8 KB are used by the bootloader

arduino due

  • ARM-based microcontroller
  • operating voltage is 3.3V
  • recommended input voltage is 7-12V. Max input voltage limit is 6-20V
  • total digital pins (r/w) are 54 of which 16 are also analog
  • 15 PWM pins
  • DC current per I/O pin is 20mA
  • DC current from the 3.3V pin is 50 mA
  • clock speed is 84 Mhz
  • SRAM is 96 KB in 2 banks, 64 + 32 KB
  • memory size is 512 KB, no memory used by the bootloader

arduino zero

same amount of pins as the Arduino UNO, but with an ARM-based microcontroller and a realtime debugger chip

arduino pro mini

  • ATmega328P microcontroller, SMD version
  • no embedded USB controller
  • 5V with 16 Mhz clock
  • 3.3V with 8 Mhz clock
